## Installation


``` 
 git clone https://github.com/sujayadkesar/Auto-recon.git
 ```
``` 
cd Auto-recon
```
``` 
chmod +x auto-recon.sh 
```
``` 
./auto-recon.sh 
```
> **Note** : To access this tool from any directory  
> ```
>  ln -sf <complete path to auto-recon.sh> /usr/local/bin/auto-recon
>  ```
